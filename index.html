About webERP

webERP is a complete web-based open-source accounting, business management and ERP (Enterprise Resource Planning) system.

webERP is provided according to the terms of the GNU General Public License v2. The current release of webERP is v4.15.2 for PHP v7. webERP v5 will be released soon with many new features and will support PHP v8.

webERP has been in active development and use for more than 20 years. The project was started by Phil Daintree and is now led by Tim Schofield, both qualified accountants, whose guiding principal has been to make webERP easy for business people to use and modify.

A wide range of features make webERP suitable for many businesses, particularly distributed wholesale, distribution and manufacturing businesses. webERP is also used by many hospitals around the world.

The webERP project also maintains webSHOP, an add-on eCommerce portal, and CounterLogicPOS, an add-on Point Of Sale portal. Use them individually or together to create a dispersed multi-branch retail management system.

This webERP project website is also open-source. You can fork weberp.org on GitHub and send a pull request with additions and improvements (please first post your ideas in a Discussions topic).
